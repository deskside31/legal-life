旧サイト（GitHub Pages）からの自動転送および案内表示のお知らせ

公開日: 2026/4/24

いつもlegal&lifeをご利用いただきありがとうございます。
この度、サイトの移転に伴い、旧アドレス（GitHub Pages）で公開していた全ページにおいて、新サイトへの移行をご案内する画面を表示するよう変更いたしました。

移行に伴う変更内容

旧アドレスにアクセスされた場合、数秒後に現在の新サイトへ自動的に移動、または案内が表示されるようになります。ブックマーク等に登録されている方は、お手数ですが新アドレスへの更新をお願いいたします。

また、サイト機能の集約とセキュリティ維持のため、GitHub Pages側の旧ソースコードに含まれていた各種機能（プログラム等）はすべて削除・停止いたしました。

対象プラットフォーム GitHub Pages（旧公開環境）

新アドレス https://legal-life.pages.dev/

今後は新環境にて、より安全で快適なコンテンツ提供に注力してまいります。
